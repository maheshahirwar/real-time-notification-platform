# Real-Time Notification Platform

A high-throughput notification platform supporting Email, SMS, and Push notifications using asynchronous event-driven architecture.

## Overview

This project demonstrates scalable notification delivery systems used by modern enterprise applications.

The platform supports reliable message delivery, retry handling, monitoring, and notification tracking.

## Features

* Email Notifications
* SMS Notifications
* Push Notifications
* Kafka-Based Messaging
* Retry Mechanism
* Delivery Tracking
* Monitoring Dashboard
* Alerting
* Notification Templates
* Bulk Processing

## Tech Stack

* Spring Boot
* Kafka
* Redis
* MySQL
* Docker
* Kubernetes

## Architecture

Producer → Kafka → Notification Service → Delivery Channels

## Notification Channels

* Email
* SMS
* Push Notifications
